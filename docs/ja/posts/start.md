# 戻ってきました~

ブログで村長のストーリーを語ったためにドメインがブロックされてしまいましたが、努力の末、ドメインをCloudflareに移管しました。

これからも自分が言いたいことを言い続けます。ここで言うことが許されないのなら、別の場所に移って言い続けるまでです。

<br>

### 2026-07-08 (追記)

このドメインは以前、技術ブログを専門に執筆するためのものであり、本来このブログで政府の不適切な点について書くつもりはありませんでした。もし私自身が同じような権力の立場に置かれたなら、やはり身内を優先し、自分や周囲の人々の利益を第一に考えてしまうかもしれないと思っているからです。

しかし、以前の記事で帰省した際の一コマを記録したことが発端となりました。党歴50年の高齢の党員から、「村の資産はすべて流出し、その多くが村の幹部個人やその親族の私有財産になってしまった」という話を聞きました。その老党員は非常に傷つき、涙を流しながら話してくれました。私はその内容をそのまま記事に記載したのです。

これを知った私たちの村の党支部書記は電話をかけてきて、記事の削除とドメインの廃止を要求しました。もし従わなければ、私が住む杭州まで来て私を拘束すると言われました。さらに、親戚一同にも電話があり、従わなければ親族全員の政治審査（身元調査）や仕事に影響を及ぼすと告げられました。その際、私の弟に対し、彼が私を説得して記事を削除させドメインを廃止させなければ、弟の2人の子供たちを学校に通わせないという具体的な脅迫がなされました。その後、彼は個人のコネとルートを使って、私の国内ドメインをブロックさせました。

私には理解できません。ただ客観的な事実を語ることも許されないのでしょうか。

この党支部書記は、**寧夏回族自治区中衛市沙坡頭区迎水橋鎮牛灘村の党支部書記、牛長春（ぎゅう・ちょうしゅん）**です。

この件の後、他の村人から彼の横暴な行為に関する情報や動画がいくつか送られてきましたが、私はそれらを一切ブログに掲載していません。それらは私自身の個人的な体験ではなく、完全に検証することができないため、掲載する必要はないと判断したからです。私はブログにおいて、自分が直接体験し、検証した事実のみを記録することにしています。
