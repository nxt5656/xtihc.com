# 我回来了

域名因为在博客上,讲我们村长的故事,被封了,经过不断的努力,把域名转移到了cloudflare.

我还是要说我想说的,这里不让我说,我就换个地方接着说。

<br>

### 2026-07-08 (补充)

我这个域名以前是专门写技术博客的，本没有计划在这个博客里写出关于任何政府做的不好的地方。我觉得，哪怕把我放到同样的位置上，在面临权力和利益时，说不定我也会任人唯亲，优先考虑自己和身边人的利益。

但是，因为我在此前的文章中提到了回老家时的一件事：一位有着五十年党龄的老党员跟我说，村里的资产已经流失完了，其中有很多变成了村领导个人或亲属的资产，老党员在说起这件事时非常伤心甚至流泪。我将这段内容记录在了文章中。

我们村的村支书知道后，打电话要求我删除文章并注销域名。他表示，如果我不按要求处理，他会从老家来杭州拘留我。同时，他给我的多位亲属打了电话，表示如果我不配合，我所有亲属的政审、工作都会受到影响。其中，他威胁我弟说，如果不能劝我把文章删除、注销域名，我弟的两个孩子都不能上学。此后，他通过个人关系和渠道，禁掉了我的国内域名。

我不明白，难道客观陈述事实也不行了吗？

这位村支书是**宁夏中卫市沙坡头区迎水镇牛滩村的村支书，牛长春**。

在此之后，也有一些村里的知情人给我发了他其他横行霸道的事和相关视频，但我都没有发到博客文章里。因为那些事并不是我本人的亲身经历，在无法完全证实的情况下，我觉得没有必要发出来。我写博客，只记录我自己亲身经历和证实的事实。
